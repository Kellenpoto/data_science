## Standards for Pca
Students should be able to
 * (MH) Describe how the principal components are constructed in PCA.
 * (M) List reasons for reducing the dimensions.
 * (M) Determine the number of principal components to keep to retain some percent of variance.
 * (M) Compute and interpret PCA using sklearn.
 * (M) Use PCA in sklearn to generate a lower-dimensional representation of data.
 * (ML) Explain the difference between PCA and t-SNE.

<br/>Priorities for Standards:
 * H:  Must achieve to graduate
 * MH: Should achieve to graduate
 * M:  Important skill though mastery not necessary
 * ML: Useful skill; encouraged to learn
 * L:  May be useful but not a focus of this program
